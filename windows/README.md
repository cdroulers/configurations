# Windows

## Setup

### On Windows

Install Alacritty
Install wsl2

### On wsl2 Ubuntu distro

Clone configurations repo.
Run `./windows/setup-symlinks.sh`
Copy alacritty folder to %APPDATA%/alacritty (for now, no symlink :()
  - Change the shell to `C:\\Windows\\System32\\wsl.exe tmux`
Install zsh
Install tmux / TPM
Install [MesloLGS NF Font](https://github.com/romkatv/powerlevel10k?tab=readme-ov-file#manual-font-installation)

### Back on Windows

Start Alacritty, everything should work!
